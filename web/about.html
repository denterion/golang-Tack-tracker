About Project

Task-Tracker FerrochiniJ is a modern, lightweight task management system designed as a full-stack learning project.

The goal is to showcase how a Go backend with REST API communicates seamlessly with a stylish frontend.

Technology Stack

  • Go
  • Echo Framework
  • PostgreSQL
  • REST API
  • JavaScript
  • HTML / CSS
