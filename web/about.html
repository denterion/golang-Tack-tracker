About Project

Task-Tracker FerrochiniJ is a lightweight task management system built as a full-stack learning project.

The goal of the project is to demonstrate how a modern backend written in Go can interact with a simple web frontend through a REST API.

Technology Stack

  • Go
  • Echo Framework
  • PostgreSQL
  • REST API
  • JavaScript
  • HTML / CSS
